# Historias de usuario implementadas

## HU-01: Registro de usuario

Como usuario, deseo registrarme en el sistema mediante mi correo electrónico y una contraseña, para poder acceder a las funciones disponibles según mi rol.

Estado: Implementada.

## HU-02: Inicio de sesión

Como usuario registrado, deseo iniciar sesión, para acceder de manera segura al sistema.

Estado: Implementada.

## HU-03: Consulta de carreras

Como estudiante, deseo consultar las carreras disponibles, para conocer la oferta académica de la universidad.

Estado: Implementada.

## HU-04: Gestión de carreras

Como administrador, deseo registrar, consultar, editar y eliminar carreras, para mantener actualizada la oferta académica.

Estado: Implementada.

## HU-05: Gestión de cursos

Como administrador, deseo registrar y administrar cursos asociados con una carrera, para organizar la oferta de asignaturas.

Estado: Implementada.

## HU-06: Gestión de docentes

Como administrador, deseo registrar y administrar docentes, para mantener actualizada la información del personal académico.

Estado: Implementada.

## HU-07: Gestión de estudiantes

Como administrador, deseo registrar y administrar estudiantes asociados con una carrera, para mantener actualizado el registro estudiantil.

Estado: Implementada.

## HU-08: Gestión de matrículas

Como administrador, deseo matricular estudiantes en cursos, para registrar su participación durante un período académico.

Estado: Implementada.

## HU-09: Validación de matrículas duplicadas

Como administrador, deseo que el sistema impida matricular dos veces al mismo estudiante en el mismo curso y período, para evitar registros duplicados.

Estado: Implementada.

## HU-10: Control de acceso por roles

Como administrador, deseo que las funciones administrativas estén restringidas por roles, para proteger la información del sistema.

Estado: Implementada.
